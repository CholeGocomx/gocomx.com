目标客户画像 · Target Customer Profiles
GoComX 核心三类目标客户 · Three Core ICP Segments
1
准品牌型工厂
⭐ 最核心人群
特征
已在 Amazon / Walmart Marketplace 有销量
想进入线下零售（Home Depot / Lowe's）
有 SKU 体系 + 条码 + 包装能力
典型行业
⚡ 电动工具 / Hand Tools 💡 Lighting / Plumbing 🏠 小家电 🌿 BBQ / Garden
痛点
被大零售选中，但卡在 EDI onboarding
不懂 850 / 856 / label / routing
💰 这类客户最愿意付钱
2
ODM / OEM 工厂
隐形大客户
特征
本来就在给美国品牌供货
想自己做品牌（DTC 或 B2B）
有产能但没渠道
痛点
不懂美国零售规则
没有 EDI 能力 → 无法直接对接
🔗 EDI 平台 = "帮他绕过中间品牌商"
3
跨境 B2B 贸易商
非铺货型
特征
已经在做批发（不是 ToC）
有海外仓，SKU 较稳定
痛点
客户要求 EDI（否则不给单）
人工处理订单成本极高
EDI 直接带来
降本（自动化）
提升接单能力
📦 EDI = 接单门票，不做就丢单
